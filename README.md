## ✨ Features

- ✅ **Single HTML file** — just open in browser, works offline
- ✅ **Syntax highlighted Python code** with one-click copy buttons
- ✅ **Interactive quiz** with answer reveal and explanations
- ✅ **Clickable checklists** to track what you've revised
- ✅ **Live countdown** to exam date (March 10, 2026)
- ✅ **Sticky navigation** with smooth tab switching
- ✅ **Fully responsive** — works on mobile too
- ✅ **No dependencies** — pure HTML/CSS/JS, no npm, no install

---

## 🛠️ How to Run Locally

1. **Download** `index.html` from this repo
2. **Double-click** the file — it opens directly in your browser
3. That's it! No server needed.
